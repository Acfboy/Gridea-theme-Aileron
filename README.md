### 简介

一个简约的Gridea主题，更详细的介绍看[这里](https://acfboy.pw/Aileron/)。


### 功能

- [x] 背景图片
- [x] 自定义透明度
- [x] 友情链接
- [x] 自定义颜色
- [x] 公告栏
- [x] 淡入动画效果
- [x] 自定义许可协议
- [x] 目录
